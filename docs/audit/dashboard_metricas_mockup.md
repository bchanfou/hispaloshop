# Dashboard de Métricas (mockup funcional recomendado)

## KPIs de negocio (home)
- GMV diario / semanal / mensual
- Conversion rate checkout
- AOV
- % pedidos con afiliado
- Comisión total plataforma vs payout productores/influencers

## KPIs operativos
- Webhook success rate Stripe
- Tiempo medio confirmación pago→orden
- Pedidos con incidencia/disputa
- Errores 5xx por endpoint

## KPIs IA
- CTR de recomendaciones
- Chat sessions activas
- Chat-to-cart conversion
- Coste OpenAI por día

## Alertas
- Picos de 429/401
- Ratio refunds > umbral
- Stockouts y oversell incidents
- Coste OpenAI > presupuesto diario
